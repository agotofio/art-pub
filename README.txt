Как залить на GitHub Pages без 404

ВАЖНО: файлы должны быть в корне репозитория (там где README.md), а НЕ внутри папки.
Залей:
- index.html
- menu.html
- gallery.html
- booking.html
- папку assets/

Если ты уже залил внутрь папки `gorynych-ABSOLUTE-FINAL`, тогда открывай:
https://<username>.github.io/<repo>/gorynych-ABSOLUTE-FINAL/index.html
(но лучше переложить в корень)

На меню/галерее: фото в модальном окне показываются полностью (object-fit: contain).
